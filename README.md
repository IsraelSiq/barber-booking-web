# 💈 Barber Booking Web

Front-end web do sistema de agendamento de barbearia, desenvolvido em **React**.

Consome a [Barber Booking API](https://github.com/IsraelSiq/barber-booking-api) — [demo ao vivo](https://web-production-e3626.up.railway.app/docs).

## 🖥️ Telas

- 🔐 Login e cadastro
- 📅 Agendamento (escolha de data e horário disponível)
- 🗓️ Meus agendamentos

## 🚀 Como rodar

```bash
# 1. Clone o repositório
git clone https://github.com/IsraelSiq/barber-booking-web.git
cd barber-booking-web

# 2. Instale as dependências
npm install

# 3. Inicie o projeto
npm start
```

Acesse em: **http://localhost:3000**
